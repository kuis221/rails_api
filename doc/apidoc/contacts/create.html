POST /api/v1/contacts
Create a new contact

Creates a new contact and returns all the contact's info, including the assigned unique ID.

Supported Formats

json, xml

Errors

406 The server cannot return data in the requested format
404 The requested resource was not found
500 Server crashed for some reason. Possible because of missing required params or wrong parameters
422 There is one or more invalid attributes for the contact

Examples

POST /api/v1/contacts?auth_token=XXXXXYYYYYZZZZZ&company_id=1
DATA:
{
    contact: {
        "first_name": "Trinity",
        "last_name": "Blue",
        "full_name": "Trinity Blue",
        "title": "MBN Supervisor",
        "email": "trinity@matrix.com",
        "phone_number": "+1 233 245 4332",
        "stree1": "1st Young st.,",
        "stree2": "2nd floor, #34",
        "city": "Toronto",
        "state": "ON",
        "country": "CA",
        "zip_code": "12345"
    }
}

RESPONSE:
{
    "id": 268,
    "first_name": "Trinity",
    "last_name": "Blue",
    "full_name": "Trinity Blue",
    "title": "MBN Supervisor",
    "email": "trinity@matrix.com",
    "phone_number": "+1 233 245 4332",
    "stree1": "1st Young st.,",
    "stree2": "2nd floor, #34",
    "street_address": "1st Young st., 2nd floor, #34"",
    "city": "Toronto",
    "state": "ON",
    "country": "Canada",
    "zip_code": "12345"
}
