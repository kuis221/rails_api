PUT /api/v1/contacts/:id
Update a contact's details

Updates a contact's information and returns all the contact's updated info.

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
406 The server cannot return data in the requested format
404 The requested resource was not found
500 Server crashed for some reason. Possible because of missing required params or wrong parameters

Examples

PUT /api/v1/contacts/268
DATA:
{
    contact: {
        "first_name": "Trinity",
        "last_name": "Blue",
        "full_name": "Trinity Blue",
        "title": "MBN Supervisor",
        "email": "trinity@matrix.com",
        "phone_number": "+1 233 245 4332",
        "stree1": "1st Young st.,",
        "stree2": "2nd floor, #34",
        "city": "Toronto",
        "state": "ON",
        "country": "CA",
        "zip_code": "12345"
    }
}

RESPONSE:
{
    "id": 268,
    "first_name": "Trinity",
    "last_name": "Blue",
    "full_name": "Trinity Blue",
    "title": "MBN Supervisor",
    "email": "trinity@matrix.com",
    "phone_number": "+1 233 245 4332",
    "stree1": "1st Young st.,",
    "stree2": "2nd floor, #34",
    "street_address": "1st Young st., 2nd floor, #34"",
    "city": "Toronto",
    "state": "ON",
    "country": "CA",
    "country_name": "Canada",
    "zip_code": "12345"
}
