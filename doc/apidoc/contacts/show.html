GET /api/v1/contacts/:id
Return a contact's details

Supported Formats

json, xml

Errors

406 The server cannot return data in the requested format
404 The requested resource was not found
500 Server crashed for some reason. Possible because of missing required params or wrong parameters

Examples

{
    "id": 268,
    "first_name": "Trinity",
    "last_name": "Blue",
    "full_name": "Trinity Blue",
    "title": "MBN Supervisor",
    "email": "trinity@matrix.com",
    "phone_number": "+1 233 245 4332",
    "stree1": "1st Young st.,",
    "stree2": "2nd floor, #34",
    "street_address": "1st Young st., 2nd floor, #34"",
    "city": "Toronto",
    "state": "ON",
    "country": "CA",
    "country_name": "Canada",
    "zip_code": "12345"
}

Params

Param name Description
auth_token
required

User's authorization token returned by login method


Value: Must be String
company_id
required

One of the allowed company ids returned by the “User companies” API method


Value: Must be a number.
id
required

Contact ID


Value: Must be a number.
