DELETE /api/v1/events/:event_id/comments/:id
Deletes a comment

Supported Formats

json, xml

Errors

404 Record not found
401 Unauthorized access
500 Server crashed for some reason

Examples

DELETE /api/v1/events/192/comments/12.json?auth_token=AJHshslaA.sdd&company_id=1
RESPONSE:
{
  success: true
  info: "The comment was successfully deleted"
  data: { }
}

Params

Param name Description
auth_token
required

User's authorization token returned by login method


Value: Must be String
company_id
required

One of the allowed company ids returned by the “User companies” API method


Value: Must be a number.
event_id
required

Event ID


Value: Must be a number.
id
required

Comment ID


Value: Must be a number.
