GET /api/v1/campaigns/:id/events
Returns a list of events for a given campaign.

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

GET: /api/v1/campaigns/1/events.json
{
  [
    [38292,"2014-06-28T13:00:00.000-07:00"],
    [36244,"2014-06-30T17:00:00.000-07:00"],
    [36812,"2014-06-30T19:00:00.000-07:00"],
    [36255,"2014-07-01T05:00:00.000-07:00"]
  ]
}
