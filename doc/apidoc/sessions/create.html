POST /api/v1/sessions
Authenticate a user and return the authentication token

Validates the user credentials and returns the authentication token if valid.

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
401 Invalid credentials
500 Server crashed for some reason

Examples

POST /api/v1/sessions.json

POST DATA: {
  email: 'fulano@detal.com',
  password: 'MySuperSecretPassword'
}

RESPONSE:
{
  sucess: true,
  info: 'Logged in',
  data: {
    auth_token: 'XXXYYYYZZZ',
    current_company_id: 1
  }
}
