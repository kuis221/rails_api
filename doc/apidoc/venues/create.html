POST /api/v1/venues
Creates a new venue

Also see venues#types.

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
401 Unauthorized access
500 Server crashed for some reason

Params

Param name Description
venue
required
Value:

Must be a Hash
venue[name]
required

The Venue name


Value:

Must be String
venue[types]
required

A comma separated string with one or more of the possible types venue types. See venues#types for more info


Value:

Must be String
venue[street_number]
required

The Venue street_number (address 1)


Value:

Must be String
venue[route]
optional , nil allowed

The Venue's route (address 2)


Value:

Must be String

required

The Venue country code. Eg. US


Value:

Must be String
venue[city]
required

The Venue city


Value:

Must be String

required

The Venue state, use the full name and not the code. Eg “California” or “Florida”
